# Audio Jones Homepage Asset Structure

Place this folder at:

```txt
/public/assets/homepage/
```

## Production Rule

- `references/` = visual guidance only.
- `backgrounds/` = atmosphere only.
- `portrait/` = foreground portrait assets.
- `icons/`, `diagrams/`, `charts/` = reusable UI assets.
- All important text must stay live in code for SEO/AEO.
- Do not flatten complete sections into images.

## Hero Rule

Build the hero as layers:

```txt
Background image
Noise layer
Live ALL SIGNAL typography
Portrait image
Signal node overlay
Live headline / body / CTA
System diagram
Metrics strip
```
